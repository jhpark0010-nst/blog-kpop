## 2. 인용문 처리 규칙

- 원문의 **별개 발언을 하나로 합치지 말 것**. 각 발언은 따로 인용 처리.
- 원문에 **없는 발언/코멘트를 생성하지 말 것** (예: "법무팀은 이에 유감을 표했다" 같이 원문에 없는 반응 추가 금지).
- 원문 발언의 **순서를 임의로 재배치하지 말 것**.
- 중요한 인용 도입부 (예: "새 앨범과 월드투어로 돌아올게요") 는 **가능하면 보존**.

---

## 3. 그룹 나열 / 소속사 맥락

- 원문에 나열된 제작 참여 그룹 / 아티스트 리스트는 **순서 그대로, 누락/대체 없이** 번역.
- 예: 원문 "투모로우바이투게더(TXT), 코르티스, 르세라핌, 엔하이픈" → `TXT, CORTIS, LE SSERAFIM, ENHYPEN` (CORTIS 누락 금지, KATSEYE 같은 외부 그룹 임의 추가 금지)

---

## 4. 팩트 정확성

- **숫자 정확히**: 차트 순위, 조회수, 발매 날짜, 멤버 수 등.
- **누락 OK / 날조 NO**: 원문에 있는 정보를 빼는 건 가능. 원문에 없는 정보를 만들어 쓰는 건 절대 금지.
- **한국 용어**: 원문에만 있는 한국어 발음/표기 (예: 음방, 팬사인회) 는 영어 독자용으로 설명 추가 (music show, fan meeting).

---

## 5. SEO 키워드

- **첫 문단**: 아티스트명 + 구체 사실 (숫자/날짜/장소) 반드시 포함.
- **Title**: 주 키워드가 앞 30자 이내.
- **Slug**: 영어 소문자 하이픈, 3~6 단어.
