## 2. 번역 충실성 — 최우선

**원칙: 번역만. 재구성/재편집/재해석 금지.**

- **문단 순서 보존**: 원문의 문단 순서를 그대로 따를 것. 영어판에서 문단을 재정렬하거나 통합/분할하지 말 것.
- **한 문단 = 한 문단**: 원문 1 문단 → 영어 1 문단 기준. 가독성상 불가피할 때만 최소한의 조정.
- **인용문 분리 유지**: 원문의 별개 발언을 하나의 연속된 speech로 합치지 말 것. 각 발언은 별도 인용.
- **인용 누락 금지**: 원문의 주요 인용 도입부 (예: "새 앨범과 월드투어로 돌아올게요") 는 보존.
- **발언 순서 유지**: 인용문 순서를 임의 재배치하지 말 것.
- **나열 완전성**: 원문이 그룹/곡/멤버를 나열하면 그대로 번역. 누락/추가/대체 금지.
- **해석 추가 금지**: "업계는 주목한다", "충격적 소식", "예상을 뒤엎고" 같이 원문에 없는 해설/프레이밍 추가 금지.
- **맥락 날조 금지**: 원문에 배경 설명이 없으면 독자 이해를 위해 추가 말 것. 원문 정보 범위 내에서만 기술.

예외: Title / Meta / 첫 문단 lead 는 SEO 상 약간의 적응 허용 (키워드 배치). 본문은 엄격히 충실 번역.

## 3. 그룹 나열 / 소속사 맥락

- 원문에 나열된 제작 참여 그룹 / 아티스트 리스트는 **순서 그대로, 누락/대체 없이** 번역.
- 예: 원문 "투모로우바이투게더(TXT), 코르티스, 르세라핌, 엔하이픈" → `TXT, CORTIS, LE SSERAFIM, ENHYPEN` (CORTIS 누락 금지, KATSEYE 같은 외부 그룹 임의 추가 금지)

---

## 4. 팩트 정확성

- **숫자 정확히**: 차트 순위, 조회수, 발매 날짜, 멤버 수 등.
- **누락 OK / 날조 NO**: 원문에 있는 정보를 빼는 건 가능. 원문에 없는 정보를 만들어 쓰는 건 절대 금지.
- **한국 용어**: 원문에만 있는 한국어 발음/표기 (예: 음방, 팬사인회) 는 영어 독자용으로 설명 추가 (music show, fan meeting).

---

## 5. SEO 키워드

- **첫 문단**: 아티스트명 + 구체 사실 (숫자/날짜/장소) 반드시 포함.
- **Title**: 주 키워드가 앞 30자 이내.
- **Slug**: 영어 소문자 하이픈, 3~6 단어.
